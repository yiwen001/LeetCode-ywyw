**209. 双指针+滑动窗口**

input: s，nums. 

a.套路判断list是否为空
```
if not nums:
   return 0
```
b.初始化一些瓶子
```
   plus=0
   left=0
   right=0#初始化右指针
   min-length=float('inf')#最小长初始化为无穷大，因为写的这个函数是针对于所有未知长度的list的
```
c.开始滑动窗口
```
   for left in range(len(nums)): #左指针移动
       while(right<len(nums) and plus<s)：
             plus+=nums[left]
             right=right+1
       if(plus>=s):
          min_length=min(min_length, right-left)
          plus=plus-nums[left]
    
 ```
 d. 判断是否存在满足条件的子数组：
 ```
    if min_length==float('inf'):#没有找到min_length
       return 0
 ```
    return min_length
